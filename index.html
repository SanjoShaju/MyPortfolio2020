Visual Inspection App

An android app build for ZF Windpower to automate their visual inspection report process.

>Business Case

There was a need to automate the report generation process and also digitalize it, as these reports was something that need to be saved and archived for at-least 10 years. Before it was a paper based report generation.

>Solution

I created an android app to solve this, which had additional features like send images of faults along with report, easily notify the required personals on some error etc.

On top of the app requirement, an automated backend was created which stored the generated reports in a shared location and also updated the data on to an excel which showed the status of the reports process on a live screen.

  • Android
  • Java
  • SQLite

Instagram bot

A python based bot to automaticaly comment and like other instagram users posts relating to a particular niche

>Business Case

I was running a comic based instagram profile named @ghot_cornered and wanted to increase social media reach to it.

>Solution

Instagram bots in the market was monotonous and the result that I got from it was not favourable to my account.

Thus I created a fully customized bot which searchs posts based on hashtags that are related to my profile, and comment on it using chat bot NLP capabilty. It also likes such posts and also follow the users too.
